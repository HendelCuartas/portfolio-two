# Personal Portfolio
Second portfolio this time made using React.js

# Install

With npm:
```
npm install
```
With yarn:
```
yarn install
```
# Development
With npm:
```
npm run dev
```
With yarn:
```
yarn dev
```
This will run at localhost:8888

# Production
With yarn:
```
yarn build
```
